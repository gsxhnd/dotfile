# Mac install dotfile

## 安装依赖

提前安装 xcode 并打开 xcode 同意许可。

## 执行安装

```bash
# 安装 omz
./install_mac.sh omz
# 安装 Homebrew
./install_mac.sh brew
# 安装 Rust <www.rust-lang.org>
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
# 配置文件
./install.sh config
```
